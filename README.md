# 💸 Splitr – Smart Bill Splitting for Roommates & Groups

**Splitr** is a modern, mobile-first web app that helps you fairly **split expenses** with friends or roommates. Whether it's rent, groceries, or trip costs—Splitr makes it simple, itemized, and transparent.

---

## 🚀 Features

- 📋 Add itemized bills (name, quantity, cost, tax)
- 👥 Assign items to specific people
- 🧮 Automatic share calculation (per item + total)
- 🏠 Organize by groups (e.g. “June Rent”, “Trip to NYC”)
- 📤 Export summaries per user (PDF, Excel, or Splitwise-style)
